# Agente Python Fixer

Projeto para correção automática de código Python utilizando agentes de IA.

## Sprint 1
- Estrutura básica do projeto.
- Módulos principais: fixer, agent, utils.
- Testes iniciais.
- Configuração com Poetry.

## Instalação

```bash
poetry install

```

## Execução

```bash
poetry run python src/main.py

```
