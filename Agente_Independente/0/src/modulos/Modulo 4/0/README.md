# Módulo 4 - Expert Adapta

Este módulo implementa o motor especialista do Adapta.

## Estrutura
- src/engine/: lógica central do expert
- src/bridge/: pontes de comunicação
- src/memory/: armazenamento de sessão e projeto
- src/validators/: validação de contexto
- src/templates/: templates de prompt
- src/identity/: identidade e políticas
- config/: configurações
- 	ests/: testes unitários
